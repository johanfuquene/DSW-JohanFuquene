FLEX BOX

Segun by MDN contributors (12demarzo2022) El Módulo de Caja Flexible, comúnmente llamado flexbox, fue diseñado como un modelo unidimensional de layout, y como un método que pueda ayudar a distribuir el espacio entre los ítems de una interfaz y mejorar las capacidades de alineación. Este artículo hace un repaso de las principales características de flexbox, las que exploraremos con mayor detalle en el resto de estas guías. Cuando describimos a flexbox como unidimensional destacamos el hecho que flexbox maneja el layout en una sola dimensión a la vez — ya sea como fila o como columna. Esto contrasta con el modelo bidimensional del Grid Layout de CSS, el cual controla columnas y filas a la vez. MDNcontributors(2022)

LISTAS DE ENLACES

Listas HTML segun el portal 2005 - 2020 ITECAD Soluciones Las listas en HTML nos permiten crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos, generalmente, por un guion o número. Los tipos de listas en HTML son los siguientes: Listas ordenadas Listas desordenadas Listas Ordenadas Las listas en HTML ordenadas son aquellas que nos muestran los elementos de la lista en orden. Para representar el orden tendremos los elementos numerados. Es decir, cada uno de los elementos irá precedido de un número o letra que establezca su orden. Las listas en HTML ordenadas se representan mediante el elemento OL.

    ...
Cada uno de los elementos de la lista ordenada se representará mediante el elemento LI.
  1. Elemento 1
  2. Elemento 2
    3. ...
  3. Elemento N
Un ejemplo de lista ordenada sería el siguiente:
  1. Julio
  2. Carmen
  3. Ignacio
  4. Elena
Que produciría el siguiente efecto: Julio Carmen Ignacio Elena Número de inicio de la lista: start El atributo start nos permite indicar el número por el cual queremos que empiece la lista, ya que por defecto las listas ordenadas en HTML empiezan por el número 1.
    ...
De esta forma, si queremos que la lista empiece por el número 5, escribiremos el siguiente código:
  1. Julio
  2. Carmen
  3. Ignacio
  4. Elena
Que produciría el siguiente efecto: 5. Julio 6. Carmen 7. Ignacio 8. Elena Listas Desordenadas Las listas desordenadas en HTML nos sirven para mostrar los elementos sin ningún tipo de orden, simplemente precedidos por una viñeta que puede ser un punto, un cuadrado,… Para definir una lista desordenada en HTML utilizamos el elemento ul.
    ...
Para representar los elementos de la lista desordenada utilizamos el mismo elemento que con las listas ordenadas, es decir, el elemento li.
  • Elemento 1
  • Elemento 2
    • ...
  • Elemento N
De esa forma podríamos tener el siguiente código HTML:
  • FC. Barcelona
  • Real Madrid
  • Real Betis
  • Atlético de Madrid
Lo que nos mostrará: FC. Barcelona Real Madrid Real Betis Atlético de Madrid listas html no ordenadas tomado de https://www.itecad.com/nuestro-blog/164-desarrollo-web/350-como-insertar-elementos-de-texto-enlaces-vinculos-y-listas-en-una-pagina-html Copyright © 2005 - 2020 ITECAD Soluciones

ENLACES

segun el portal 2005 - 2020 ITECAD Soluciones, Antes de crear enlaces en HTML debemos de tener presente que es una URL y cuáles son sus partes. URL son las siglas en inglés de Uniform Resource Locator, que en español significa Localizador Uniforme de Recursos. Como tal, el URL es la dirección específica que se asigna a cada uno de los recursos disponibles en la red con la finalidad de que estos puedan ser localizados o identificados. Así, hay un URL para cada uno de los recursos (páginas, sitios, documentos, archivos, carpetas) que hay en la World Wide Web. El URL es usado para encontrar un recurso en Internet. Por ejemplo si quieres buscar un artículo en la página 'itecad.com' sabes que el URL que te va a llevar a él es https://www.itecad.com. Esta dirección se compone de: 'https://' que sería el protocolo de acceso para las páginas de internet. 'www' que es la dirección del recurso. '.itecad' sería el nombre del dominio y '.com' es el tipo de dominio. Los hipervínculos, enlaces o links son aquellas palabras o imágenes que tienen un URL ligada a ella y al hacer clic sobre estos el navegador dirige al usuario al sitio o página enlazado. En HTML los enlaces se marcan con la etiqueta< a>/a> y el atributo principal es href="/" donde se escribe la ubicación del archivo de destino que puede estar en la misma carpeta que el archivo que lo está llamando, en otra carpeta del mismo sitio o en otro sitio web.
